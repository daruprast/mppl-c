4. Sebutkan dan jelaskan jenis version control system yang digunakan dalam pengembangan perangkat lunak.
- Local Version Control
Dalam version control ini, file hanya disimpan secara lokal di komputer
-Centralized Version Control
Dalam Centralized version control, seluruh perubahan dan versi kode disimpan menggunakan server pusat. 
Programmer harus mengakses ke server pusat sehingga server sedang down maka akses ke kode akan terhenti.
Contoh : SVN dan CSV
-Distributed Version Control 
Dalam version control ini, programmer dapat memiliki repositeri secara lengkap termasuk seluruh riwayat perubahan. 
Sehingga programmer tidak bergantung dengan server pusat.
Contoh : Git dan Mercurial
