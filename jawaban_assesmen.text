Kelebihan Metode Agile:
1. Fleksibel dan mudah adaptasi
2. Komunikasi lebih baik
3. Pengiriman lebih cepat dan bertahap
4. Kualitas produk lebih baik
5. Pelanggan lebih terlibat

- Tugas Scrum Master:
1. Memastikan proses Scrum berjalan lancar
2. Membantu tim memahami dan menerapkan Scrum
3. Menghapus hambatan kerja (impediments)
4. Melindungi tim dari gangguan eksternal

- Pengertian WIP (Work In Progress) di Kanban:
Batasan jumlah pekerjaan yang sedang dikerjakan untuk menghindari multitasking berlebihan dan meningkatkan fokus.

- Jenis Version Control System:
1. Centralized VCS (CVCS): Semua versi disimpan di server pusat, misalnya Subversion (SVN).
2. Distributed VCS (DVCS): Setiap pengembang memiliki salinan penuh repositori, misalnya Git dan Mercurial.
